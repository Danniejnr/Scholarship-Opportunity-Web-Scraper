# Scholarship Opportunity Web Scraper

## Overview

This project automates the collection of international scholarship opportunities from publicly available scholarship websites using Python web scraping techniques.

The scraper extracts scholarship information such as title, country, degree level, funding type, application deadline, and application link, then stores the data in a structured format for analysis and future use.

The project demonstrates practical data collection skills that form the foundation of many data science workflows.

---

## Objectives

- Collect scholarship information from online sources
- Extract relevant scholarship details
- Store scraped data in CSV format
- Handle multiple pages of results
- Clean and organize collected data
- Build a reusable scholarship discovery tool

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Scholarship-Opportunity-Web-Scraper.git
```

Install required packages:

```bash
pip install -r requirements.txt
```

---

## Usage

Run the scraper:

```bash
python scraper.py
```

The output dataset will be saved inside:

```text
data/scholarships.csv
```
